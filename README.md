# prodos
virtual prodos for linux

# usage
To launch the program type something like:
```
./prodos BASIC.SYSTEM

Usage: prodos [args] sysfile
   [-r directory] Root directory of Prodos filesystem
   [-w directory] Set working directory
   [-p port] Set serial port
   [-t] Trace execution
   [-s] Silent mode
```
